# Contributing

Thanks for your interest in contributing!

## Development
1. Install Rust (stable).
2. Build:
   ```bash
   cargo build
   ```
3. Run:
   ```bash
   cargo run
   ```

## Tests
```bash
cargo test
```

## Formatting
```bash
cargo fmt
```

## Pull Requests
- Keep changes focused and small.
- Update docs where relevant.
- Include a changelog entry when behavior changes.
