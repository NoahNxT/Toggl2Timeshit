# Settings

Open Settings with `s`.

Settings are organized into categories (left column). Use `Up/Down` to select a category and `Enter` to edit.

## General
- **Target hours**  
  Used to color the footer total (green if met, red if below).

## Integrations
- **Toggl token**  
  Update the API token from inside the app. The token is stored at:
  ```
  ~/.toggl2tsc
  ```

### Editing
- `Enter`: Start editing / Save
- `Esc`: Cancel / Back to categories

### Config File
Settings are stored in:
```
~/.toggl2tsc.json
```
