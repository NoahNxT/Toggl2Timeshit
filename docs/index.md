# Timeshit TUI Documentation

Timeshit is a Rust-based Terminal UI (TUI) that turns Toggl Track time entries into a clean, navigable timesheet dashboard. It is optimized for free-tier API limits using a persistent cache and manual refresh.

## Highlights
- Modern TUI with light/dark themes
- Date range filtering and fast navigation
- Clipboard export for timesheet tools
- Persistent cache to reduce API calls
- Built-in settings for target hours and integrations
